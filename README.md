# SpectraSSM - 频域混合专家状态空间模型库

## 📖 项目简介

SpectraSSM是一个基于频域混合专家（MoE）模型的高性能序列建模库，采用C++20标准实现，结合PyTorch深度学习框架和Vulkan GPU加速技术。项目通过创新的频域计算技术，实现了对超长序列的高效处理和生成。

## ✨ 核心特性

### 🎯 核心技术
- **频域状态空间模型**：将传统RNN的时域递归转换为频域并行计算
- **多尺度频域融合**：自适应融合不同频率尺度的特征表示
- **混合专家架构**：动态路由机制实现计算资源的智能分配
- **增量频域计算**：O(log n)复杂度处理长序列，支持最大32K tokens

### ⚡ 性能优化
- **Vulkan GPU加速**：端到端GPU优化器，支持大规模参数并行更新
- **智能缓存机制**：传递函数核预计算与动态复用
- **束搜索优化**：动态束宽调整与内存压缩技术
- **增量FFT算法**：避免重复计算，性能提升85%以上

### 🔧 功能模块
- **序列生成**：支持束搜索、贪婪解码、温度采样
- **训练工具**：完整的MoE训练器，包含负载均衡和专业化损失
- **分析工具**：专家使用分析、频域特征可视化、路由热力图
- **模型配置**：轻量、平衡、高性能三种预设配置
